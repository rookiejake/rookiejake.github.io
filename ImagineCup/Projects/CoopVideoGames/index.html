Cooperation in Video Games

By Naomi Fiebelkorn
April 2018

Overview

The purpose of this project is to evaluate cooperative design patterns by designing, implementing, and playtesting a video game. This project addresses issues in the gaming industry including the debate over violence in video games and its effects on adolescents, as well as the negativity that arises in social interactions of competitive online multiplayer games.


Objectives

The goals of this project are the following:

  • Identify patterns of cooperative game design
  • Implement simple modules using these patterns
  • Design and build a cooperative game using these modules
  • Design and conduct playtesting to gain feedback on user experience
The learning objective of this project is to evaluate the effectiveness of cooperative attributes in game design.
